## Структура проекта

    - config.py — настройки и пути к данным.
    - data_loader.py — загрузка котировок (yfinance) и расчет дневной/овернайт доходности.
    - correlation.py — вычисление матриц кросс-корреляций.
    - analysis.py — анализ и визуализация сетей лидеров и лаггеров рынка.
    - clustering.py — алгоритмы кластеризации для выявления лидеров и лаггеров рынка.
    - experiment_cv.py — тестирование модели на разных размерах тренировочной и тестовой выборки
    - model.py — генерация фичей, формирование catboost модели и подбор гиперапараметров на оптимальных рахмерах train, test
    - portfolio.py — основной пайплайн: генерация ML-фичей, обучение CatBoost и walk-forward тестирование.

    - data
        - tickers.txt - список тикеров s&p500 + сектор +индустрия
        - data_3_0.parquet - данные тикеров, загруженные с yahoofinance
        - corr_dt_ol, corr_ol_dt - рассчитанные матрицы корреляций
